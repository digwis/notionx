First public release of `create-nextion-app` to npm. Scaffolds a new vinext project that consumes `@nextion/core`, with optional one-step provisioning of Cloudflare (D1 / KV / R2 / Turnstile), Notion (data source + seed pages), Resend, and Google OAuth. Run with `npx create-nextion-app` or `pnpm create nextion-app`.
